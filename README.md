# 강화학습 기초부터 응용까지
🤖 강화학습 초짜들이 만나 정리한 스터디 기록

강화학습의 'ㄱ' 자도 모르는 4명이 만나 방학동안 공부한 내용을 기록합니다. 처음 강화학습을 공부하는 사람들도 이해하기 쉽게 내용을 정리하고자 합니다. `프로그래머를 위한 강화학습` 교재를 기반으로 하며, 다른 여러 학습자료를 활용하여 내용을 구성하였습니다. 

* **Period** : 2024/01/09 - 2024/02/26 (두 달)
